## 項目類型（ItemType）
```js
const zoteroPane = Zotero.getActiveZoteroPane();
const selected = zoteroPane.getSelectedItems();
for (const item of selected) {
  item.setType(Zotero.ItemTypes.getID('此處修改項目類型'));
  await item.saveTx();
}
```
