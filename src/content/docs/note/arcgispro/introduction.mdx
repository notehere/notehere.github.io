## 安裝前準備
<Steps>

1. 查看ArcGIS Pro的系統需求

   網址：https://pro.arcgis.com/en/pro-app/latest/get-started/arcgis-pro-system-requirements.htm

2. 下載安裝.NET Desktop Runtime 8.0

   網址：https://dotnet.microsoft.com/en-us/download/dotnet


</Steps>

## 安裝ArcGIS Pro

<Steps>

1. 登入ArcGIS Online

   網址：https://www.arcgis.com

2. 進入My Esri頁面

   在導航欄選擇「下載」，然後進入ArcGIS Pro下載頁面。

3. 下載安裝程式

</Steps>
